## Features (Version 1)

- **Single QR Code** that opens the website
- **5 Action Icons** in Olympic-style layout:
  - Top row: Instagram, Payment, WhatsApp (3 icons)
  - Bottom row: Website, Google (2 icons, centered)
- **ADX Logo** at the top
- **Footer** with "Built by AI Developer India" and clickable logo
- **Mobile-first responsive design**
- **Vercel deployment ready**

## To Restore/Run Version 1

1. Copy all files from `version1/` to a new directory
2. Run `npm install` to install dependencies
3. Run `npm start` to start the server
4. Visit `http://localhost:4000`
