To remove the limit, you need to add the 'aid' (App ID) parameter from your bank/payment gateway.

Parameters Explained:
=====================
- pa = Payee Address (your UPI ID) - REQUIRED
- pn = Payee Name (display name) - REQUIRED
- am = Amount in rupees - OPTIONAL (if not provided, user enters amount)
- cu = Currency (INR for Indian Rupees) - OPTIONAL (defaults to INR)
- aid = App ID (for merchant payments) - OPTIONAL but CRITICAL to remove ₹2000 limit
- tn = Transaction note - OPTIONAL
- tr = Transaction reference - OPTIONAL

How to Get 'aid' Parameter:
===========================
1. Contact your bank's UPI merchant services
2. Register as a merchant with NPCI
3. If using payment gateway (Razorpay, Paytm), they provide the 'aid'
4. The 'aid' is unique to each merchant/business
